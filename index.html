Revolver by jmhobbs

Let users search your GitHub project issues from your website.

Revolver lets you embed a simple feedback tab on your webpage that will let users search for issues on GitHub without leaving the page.

You can try it by clicking on the "Feedback" tab on the left.

Revolver is a very young project, so expect features and appearance to get better over time.

Dependencies

  • jQuery
